# boilercontrol-node

Node.js wrapper for boiler control

# Dependencies

Requires:

- node-gyp

Requires the following installed as dynamic libraries:

- [wiringPi](http://wiringpi.com/download-and-install/)
- [boilercontrol](https://github.com/rossharper/boilercontrol)

# Build

    node-gyp configure
    node-gyp build
